Welcome to Ranking app's documentation!
=======================================

**Ranking** is a CLI App intended to calculate the ranking table for a league.

Input/output
------------
The input will be text, it is composed of 2 teams with their names and goals 
and separated by a comma like this: "Lions 3, Snakes 3".


The rules
----------
In this league, a draw (tie) is worth 1 point and a win is worth 3 points. A loss is worth 0 points.
